Работы

  • Онлайн магазин пиццы. В работе применяются React + Hooks и Redux. Для маршрутизации использован React Router v5. Данные загружаются из JSON файла с помощью Fetch. На главной странице можно отсортировать пиццы по популярности, цене, и алфавиту. В корзине автоматически подсчитывается общая сумма. Также в ней можно добавлять или удалять товар.

  • Справочник персонажей из вселенной Star Wars. Информация для проекта грузится с сайтов swapi.dev и starwars-visualguide.com. Используемые технологии: React, React Hooks, Redux ToolKit, React Router v6, Fetch, CSS-Modules, Lodash.Debounce. В справочнике реализована навигация и поиск по персонажам. Можно добавлять любимых героев в закладки. Закладки сохраняются в LocalStorage.

  • В рамках проекта было разработана полноэкранная прокрутка. Настроена валидация формы, аккордеон и слайдер. Функционал реализован на JQuery

  • Для данной работы написано модальное окно, в котором автоматически меняется описание программы тренировки в зависимости от выбора пользователя. На странице настроено два слайдера и воспроизведение видео. Все осуществлено с помощью JavaScript.

  • Сайт — сборник игр, на JavaScript. На данный момент представлены крестики-нолики и змейка. В каждой игре есть 2 уровня сложности. При создании логики в крестиках-ноликах реализован алгоритм “Минимакс”. Графическая часть змейки написана посредством canvas.

  • Дипломный проект курса “Веб-верстка для начинающих 2.0” Skillbox. В работе реализована форма обратной связи, отправка данных на почту, слайдер и адаптация страницы под мобильные устройства с помощью JQuery.
